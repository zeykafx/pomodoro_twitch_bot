# Pomodoro GO bot

Commands:
- `[prefix]pomo [time] [task]`
- `[prefix]pomo end`
- `[prefix]pomo check`

[comment]: <> (- `[prefix]pomo leaderboard <- replies the top 3 users in minutes studied`)


# instructions

Make sure to create the `.env` file with your oauth token, prefix, and channel like
```
    TOKEN="OAUTH TOKEN"
    PREFIX="PREFIX"
    CHANNEL="CHANNEL"
```
